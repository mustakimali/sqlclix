# SQLiteX

A simple SQLite database browser with TUI interface.

## Install

```bash
make publish  # builds and copies to ~/bin/
```

## Usage

```bash
sqlitex database.db
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Tab` | Switch panels |
| `F5` | Execute query |
| `F2` | New tab |
| `Ctrl+W` | Close tab |
| `Enter` | Select/expand table, view cell detail |
| `s/c/d` | Generate SELECT/COUNT/DESCRIBE query |
| `?` | Help |
| `q` | Quit |
